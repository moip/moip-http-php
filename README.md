## Dependências

* PHP >= 5.4

## Instalação

#### Usando composer

Adicione o trecho abaixo em seu arquivo `composer.json`:

    {
        "require" : {
            "moip/moip-http-php" : "dev-master"
        }
    }
    
Execute:

    composer install
